### テーマを選んだ理由
・調理師だった経験で、レシピをExelで管理するのが手間だったので、
　画像からテキスト読み込み機能があるものが欲しかったので。
・一緒にカロリーなども計算出来るものは、他にはあまり無いはず。
・レシピの登録を投稿扱いにしているので、他の人の物も検索でき、
　検索されます。
・登録者(投稿者)はdefaultで非公開。公開も選べる仕様。
